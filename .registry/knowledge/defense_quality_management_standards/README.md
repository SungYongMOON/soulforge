Public-safe routing entry for the 2026-06-15 defense quality management
standards corpus.

private RAG navigation, and LLM wiki source-list preparation. It does not copy
standard bodies, source-text chunks, NotebookLM answers, or private payloads.

- Route questions to the prepared source family and source-card manifests.
- Start private sourcebound reviews from the validated source-text indexes.
- Track HWP, paid-license, controlled-access, and internal-source blockers.
