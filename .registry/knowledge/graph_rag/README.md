# .registry/knowledge/graph_rag

- `knowledge.yaml` registers public-safe, source-supported reusable knowledge for GraphRAG / graph-assisted RAG.
- Scope is orientation, query-routing, source packet planning, and graph/vector retrieval comparison.
- It does not approve Soulforge implementation, production architecture, benchmark superiority, private corpus suitability, source truth, ontology acceptance, or NotebookLM answer authority.
- Detailed run evidence stays in the private `_workmeta` owner scope; this public entry records only reusable metadata and public source refs.
