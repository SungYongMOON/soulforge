# .registry/tools/oring_selection_calculator

- `oring_selection_calculator/tool.yaml` is the canonical tool entry for first-pass O-ring candidate screening.
- The tool is allowed to compute and compare squeeze, installed stretch, and gland fill for a user-provided geometry.
- The tool is not allowed to approve final O-ring design or replace manufacturer catalogs, official size standards, tolerance analysis, extrusion-gap review, compound-specific media compatibility, or owner engineering judgment.
- Actual workbook files stay outside the public registry as workspace artifacts; `_workmeta` records path, size, hash, and validation status.
