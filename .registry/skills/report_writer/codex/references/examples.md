# Report Writer — 채워진 예시

아래 예시는 모두 **합성(synthetic)**이다. 실제 수치/자료가 아니라 실제 전문 보고서의 구조를 모사한
연습용 견본이다. 비공개 업무 원문은 넣지 않는다.

---

## 예시 A — 실험/시험보고서 (합성)

대상: 송신센서 송신감도 시험. 구조 근거: KRISS TR-2022-018(불확도 표기), NIST TN 2077(추적성).

**So-What 5줄 (첫 장)**

- ① 왜: 송신센서 시제가 규격 송신감도를 만족하는지 미확인 → 통합시험 진입 가부를 못 정함.
- ② 뭘: 송신센서 시제 3개, 음향수조, 75 kHz 송신감도, 각 5회 측정.
- ③ 뭘얻: 평균 132 dB re 1μPa/V (범위 131~133), 확장불확도 ±1.5 dB (k=2, 95%).
- ④ 그래서: 규격 130 dB 이상을 3개 모두 충족. **최종 판정: 적합.**
- ⑤ 다음: 통합시험 진입 권고. 저온(-5℃) 감도는 미확인 → 12월 2차 시험에서 확정.

**본문 골격**

```text
1. 시험 목적   송신센서 송신감도의 규격(≥130 dB) 충족 여부 확인
2. 시험 조건   대상: 시제 3개 / 환경: 음향수조 20℃ / 장비: 교정유효 2026-08
               시험일자: 2026-05-12 / 시험자: 홍길동
3. 검토 결과   (표)
   | 항목 | 측정값(±U, k=2) | 기준 | 판정 |
   | 송신감도(75kHz) | 132 ± 1.5 dB | ≥130 dB | 적합 |
4. 종합 판정   3개 시제 모두 기준 충족. 최종 판정: 적합.
5. 고려사항    가정: 단주파 기준. 한계/미확인: 저온 감도 미측정.
6. 후속 조치   통합시험 진입(P1) / 저온 2차 시험 12월(P2)
```

## 예시 B — 분석/트레이드 보고서 (합성)

대상: 해상도 검증용 시험표적 형상 선정. 구조 근거: NREL TP-6A20-73222(비교 후 단일 선정),
FHWA SEITS(기준·가중치를 대안보다 먼저 확정).

**So-What 5줄**

- ① 왜: 시험표적 형상 미정 → 수조시험 착수 불가.
- ② 뭘: 후보 3종을 음향 반사 안정성·영상 영향·제작성 기준으로 비교(기준·가중치 사전 확정).
- ③ 뭘얻: 비교표(아래).
- ④ 그래서: **Solid Sphere 최종 선정** (반사 안정성 최고, 제작 용이).
- ⑤ 다음: Solid Sphere로 표적 제작 권고. 제작 착수 **승인 요청**(결정: PM, 기한: 2026-05-20).

```text
| 기준(가중)        | Solid Sphere | Hollow Sphere | Corner Reflector |
| 반사 안정성(0.5)  | 상           | 중            | 중               |
| 영상 영향(0.3)    | 적음         | 적음          | 큼               |
| 제작성(0.2)       | 용이         | 보통          | 어려움           |
| 종합              | 선정         | -             | -                |

권고 → 의사결정: 따라서 Solid Sphere 채택 권고. 표적 제작 착수 승인 요청
(결정권자: PM / 기한: 2026-05-20 / 미승인 시: 수조시험 일정 2주 지연).
```

## 예시 C — 데이터 덤프 → 결론형 (Before / After)

네가 늘 지적하는 "측정 자료만 붙이고 결론 없음"을 고치는 핵심 대비다.

**Before (데이터 덤프 슬라이드)**

```text
제목: 송신센서 음향시험 결과
- 송신감도 측정 그래프 3개
- 75 kHz / 100 kHz 데이터
- 수조 시험 사진
(→ "그래서 이 시험 왜 했어요? 결론이 뭐예요?")
```

**After (결론형 — 헤드라인이 결론)**

```text
헤드라인: 송신감도 132 dB로 규격(130 dB) 충족 → 통합시험 진입 가능
- 근거: 시제 3개 모두 131~133 dB (±1.5 dB, k=2, 95%)
- 단서: 저온(-5℃) 감도 미확인 → 12월 2차 시험에서 확정
- 요청: 통합시험 일정 승인 (결정: PM, 기한 2026-05-20)
```

차이: After 는 ④그래서(판정)와 ⑤다음(요청)이 헤드라인에 있다. 같은 데이터, 결론이 보임.
