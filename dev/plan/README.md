# dev/plan

## 목적

- `plan/` 은 Soulforge 개발 과정의 수행 전 계획 문서를 둔다.
- owner 전용 구조 문서와 분리해서 저장소 공용 개발 계획을 관리한다.

## 포함 대상

- 작업 계획 문서
- relocation 계획과 수정 계획
- 수행 후 체크 상태가 갱신되는 실행 체크리스트
- 런타임 연동, 프로토콜 비교, 오케스트레이션 브리지 구상 계획

## 제외 대상

- 수행 후 결과 로그
- 저장소 정본 계약 문서
- 프로젝트 전용 계획
