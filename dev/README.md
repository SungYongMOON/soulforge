# dev

## 목적

- `dev/` 는 저장소 개발 과정에서 남기는 운영용 개발 문서와 기록을 둔다.

- `plan/` 아래의 개발 계획 문서

## 제외 대상

- 저장소 정본 구조 계약 문서
- `.agent`, `.agent_class`, `_workspaces` owner 문서
