# dev

## 목적

- `dev/` 는 저장소 개발 과정에서 남기는 운영용 개발 문서와 기록을 둔다.
- 정본 구조 계약 문서와 분리해서, 작업 이력과 보조 개발 기록을 관리한다.

## 포함 대상

- `log/` 아래의 개발 이력 문서
- `plan/` 아래의 개발 계획 문서
- `experiments/` 아래의 폐기 가능한 실험 harness 와 전용 lab
- closeout 성격의 회고 문서와 변경 기록

## 제외 대상

- 저장소 정본 구조 계약 문서
- `.agent`, `.agent_class`, `_workspaces` owner 문서
