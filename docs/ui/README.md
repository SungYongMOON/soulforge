# docs/ui

## 목적

- `docs/ui/` 는 renderer v1 의 구조, 계약, selection model, theme 을 root owner 관점에서 고정한다.
- `docs/architecture/` 가 source map 과 derive 규약을 소유하면, 여기서는 renderer consumer layer 의 정본을 관리한다.

## 포함 대상

- renderer boundary 문서
- UI state contract 문서
- selection model 문서
- Adventurer's Desk theme 문서
- renderer 구현 계획 문서

## 제외 대상

- `.agent`, `.agent_class`, `_workspaces` 정본 구조 설명
- `derive-ui-state` producer 구현 세부
- legacy `ui/viewer/` 구현 설명
