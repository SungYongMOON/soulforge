# bindings

- 이 경로는 tracked example binding set 을 둔다.
- 각 파일은 contract.yaml 에서 상대 경로로 가리키는 split binding 문서다.
- `execution_profile_binding.yaml` 은 workflow step 의 execution profile ref 를 model, attached skill, MCP/tool hint 로 resolve 하는 example 이다.
- `skill_execution_binding.yaml` 은 canonical skill id 를 installed Codex skill name 으로 resolve 하는 example 이다.
