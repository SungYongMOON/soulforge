# Soulforge Report Format Example

This folder contains public-safe report templates only.

- `soulforge_report.template.md` is the canonical source-of-truth report
  template.
- `soulforge_report.template.html` is the derived standalone HTML companion
  template.

Do not put real project reports, raw files, private source text, host-local
paths, mail bodies, or protected attachment content in this example folder.
