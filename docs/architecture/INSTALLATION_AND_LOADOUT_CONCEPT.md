# 설치와 로드아웃 개념

## 설치

Soulforge는 클래스 콘텐츠를 설치 가능한 모듈로 다룬다.

설치 가능한 구성은 다음을 포함한다.

- 스킬
- 도구
- 워크플로우
- 지식 팩
- 클래스 문서

## 로드아웃

`loadout.yaml` 은 설치된 클래스에서 현재 활성화된 장비 구성을 정의한다.

최소한 로드아웃은 다음을 식별해야 한다.

- 활성 클래스
- 활성 프로필
- 장착된 스킬
- 장착된 도구
- 장착된 워크플로우
- 장착된 지식 팩
- 워크스페이스 바인딩

## 설계 규칙

설치는 무엇을 사용할 수 있는지를 설명한다.
로드아웃은 무엇이 현재 장착되어 있는지를 설명한다.
