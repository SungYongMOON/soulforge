# docs/architecture

## 목적

- `docs/architecture/` 는 저장소 전체 구조 원칙과 메타 규약을 둔다.
- 루트 README가 다루지 않는 구조 기준 문서를 이곳에서 관리한다.

## 포함 대상

- 목표 트리, 문서 소유 원칙, 저장소 목적
- 워크스페이스와 `.project_agent` 같은 저장소 공용 구조 규약
- 저장소 전체 결정 기록과 마이그레이션 참고 문서

## 제외 대상

- `.agent` 전용 구조 문서
- `.agent_class` 전용 구조 문서
- 특정 프로젝트 전용 문서
