# .agent_class/tools/local_cli

## 목적

- `local_cli/` 는 호스트 로컬 CLI 기반 도구 래퍼를 둔다.
- 외부 연결 정의와 분리해서 로컬 실행 바인딩만 관리한다.

## 포함 대상

- 로컬 CLI 실행 래퍼
- 명령 바인딩과 실행 메타

## 제외 대상

- 원격 인증 연결 정의
- MCP 서버 프로토콜 바인딩
