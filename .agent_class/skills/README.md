# .agent_class/skills

## 목적

- `skills/` 는 클래스가 익힌 행동 패턴을 둔다.
- 도구 연결이나 지식 저장과 분리된 실행 습관 계층을 관리한다.

## 포함 대상

- 설치된 스킬 정의
- 스킬별 메타와 호출 단위 설명

## 제외 대상

- 외부 도구 연결 구현
- 절차 전체를 묶는 워크플로우와 지식 팩
