# .agent_class/docs

## 목적

- `.agent_class/docs/` 는 `.agent_class` legacy bridge 문서를 둔다.
- active owner-local 구조 문서는 `.registry/docs/architecture/` 를 기준으로 본다.

## 포함 대상

- `architecture/`, `plans/`, `prompts/`
- cleanup 전까지 필요한 legacy 안내와 계획 문서

## 제외 대상

- 저장소 전체 구조 문서
- 프로젝트 전용 문서와 본체 owner 문서
- 저장소 공용 개발 계획과 개발 이력 문서
