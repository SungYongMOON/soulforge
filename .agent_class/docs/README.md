# .agent_class/docs

## 목적

- `.agent_class/docs/` 는 class owner 문서를 둔다.
- 루트 `docs/` 와 분리해서 클래스 구조, 계획, 로그, 프롬프트를 관리한다.

## 포함 대상

- `architecture/`, `plans/`, `devlog/`, `prompts/`
- 클래스 소유 설명 문서와 운영 기록

## 제외 대상

- 저장소 전체 구조 문서
- 프로젝트 전용 문서와 본체 owner 문서
