# .agent_class/docs/prompts

## 목적

- `prompts/` 는 class 개발과 문서 패치에 반복해서 사용할 프롬프트 자산을 둔다.
- 재사용 가능한 클래스 계층 프롬프트만 모은다.

## 포함 대상

- 재사용 프롬프트 자산
- `PROMPT_<topic>.md` 형식의 프롬프트 파일
- class 개발, 문서 패치, 정리 작업용 프롬프트

## 제외 대상

- 실행 로그와 수행 계획
- 프로젝트 전용 프롬프트와 로컬 상태 메모
