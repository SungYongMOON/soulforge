# stats

- What belongs here: sanitized chain-level fit notes, routing observations, and public-safe tuning summaries for knowledge ingest.
- What does not belong here: copied source text, unlocked files, password values, private wiki projections, raw run truth, source file hashes, runtime absolute paths, NotebookLM answers, or project payloads.
- How to use it: keep evidence-bearing run outputs under `_workmeta/<project_code>/runs/<run_id>/` or `_workmeta/system/runs/<run_id>/`, then promote only public-safe summaries here after review.
