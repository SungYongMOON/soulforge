# stats

- What belongs here: public-safe routing observations, reusable SE assistant
  chain-fit notes, and sanitized tuning summaries.
- What does not belong here: project source payloads, HWP/HWPX body content,
  private run truth, source file hashes, NotebookLM answers, mail bodies,
  credentials, or runtime absolute paths.
- How to use it: keep evidence-bearing run outputs under `_workmeta/**`, then
  promote only reviewed public-safe summaries here.
