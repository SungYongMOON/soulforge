# stats

- 무엇을 둔다: Lineage Strike template-level fit summary, sanitized observation, reusable tuning note.
- 무엇을 두지 않는다: raw lineage source dump, run id, project-specific performance metrics, feedback transcript.
- 왜 이렇게 둔다: stats 는 lineage-production 조합의 관찰 요약만 남기고 실제 run truth 는 `_workspaces/` 에 남겨야 한다.
