# tools

## 목적

- `tools/` 는 renderer 와 root-owned 검증용 보조 도구를 둔다.
- 앱이나 reusable package 바깥에서 UI 정합성 검사와 optional integration utility 를 관리한다.

## 포함 대상

- `ui-lint/`
- `legacy-python-viewer/`

## 제외 대상

- canonical source 의 정본 정의
- renderer runtime package
