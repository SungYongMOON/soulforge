# 던전 배경 (달빛조각사 스킨) — 로컬·비공개

판타지 모드에서 **과제 허브에 들어가면** 그 과제의 배경 이미지가 화면에 깔립니다(과제마다 다른 "던전").

## 넣는 법
이미지를 아래 경로에 **과제번호.jpg** 로 두면 됩니다:

```
ui-workspace/apps/dev-erp/static/skins/dungeons/<과제번호>.jpg
```

- 예: 과제 `P26-005` → `static/skins/dungeons/P26-005.jpg`
- 권장: 가로형(예 1600×900 이상), `.jpg`.
- 파일이 없으면 그냥 미스트 그라데이션 배경(현행)으로 보입니다.
- 위에 옅은 크림 스크림이 깔려 패널/글자는 그대로 읽힙니다(가독성 유지).
