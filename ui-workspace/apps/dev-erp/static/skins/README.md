# 던전 배경 (달빛조각사 스킨) — 로컬·비공개

판타지 모드에서 **과제 허브에 들어가면** 그 과제의 배경 이미지가 화면에 깔립니다(과제마다 다른 "던전").

## 넣는 법
판타지 모드 기준, 두 가지 배경:

| 화면 | 파일 경로 |
|---|---|
| **메인(콕핏/프로젝트 홈)** = 길드 내부 전경 | `static/skins/main.jpg` |
| **과제 허브** = 그 과제의 던전 | `static/skins/dungeons/<과제번호>.jpg` |

- 예: 과제 `P26-005` → `static/skins/dungeons/P26-005.jpg`, 메인 → `static/skins/main.jpg`

**기본 배경(원본 일러스트)**: 이미지를 안 넣어도 `skins/regions/*.svg`(직접 그린 원본, 저작권 무관)가 깔립니다.
owner가 위 경로에 jpg를 넣으면 그게 **우선**(레이어 위)으로 덮입니다. (jpg 없으면 svg → 그라데이션 순서)
- 권장: 가로형(예 1600×900 이상), `.jpg`.
- 파일이 없으면 그냥 미스트 그라데이션 배경(현행)으로 보입니다.
- 위에 옅은 크림 스크림이 깔려 패널/글자는 그대로 읽힙니다(가독성 유지).
