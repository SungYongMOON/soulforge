## summary
개인 캘린더 export: due 있는 할일을 RFC5545 VEVENT(종일)로 묶어 .ics 다운로드. gateway 의 .ics 문자열 패턴을 '복사'(import 금지) — store.mjs 순수함수 icsForItems(items)(node:* 만). server.mjs GET /api/calendar.ics(text/calendar + attachment). app.js 일정/홈에 'ICS 내려받기' a[download]. due·title·project_id 메타만(secret 0).

## data_model
- 테이블 0. store.icsForItems(items,{calName}) → RFC5545: BEGIN:VCALENDAR/VERSION:2.0/PRODID + 각 item due 를 DTSTART;VALUE=DATE:YYYYMMDD 종일 VEVENT(UID=item.id@dev-erp, SUMMARY=esc(title), DESCRIPTION=project_id) + END:VCALENDAR. CRLF, 텍스트 이스케이프(쉼표·세미콜론·역슬래시·개행).

## code_changes
- src/store.mjs icsForItems(items,{calName='dev-erp'}={}) { const esc=(s)=>String(s??'').replace(/([,;\\])/g,'\\$1').replace(/\n/g,'\\n'); const lines=['BEGIN:VCALENDAR','VERSION:2.0','PRODID:-//dev-erp//KO','CALSCALE:GREGORIAN','METHOD:PUBLISH','X-WR-CALNAME:'+esc(calName)]; for(const i of items){ if(!i.due) continue; const dt=String(i.due).replace(/-/g,''); lines.push('BEGIN:VEVENT','UID:'+i.id+'@dev-erp','DTSTART;VALUE=DATE:'+dt,'SUMMARY:'+esc(i.title),'DESCRIPTION:'+esc(i.project_id||''),'END:VEVENT'); } lines.push('END:VCALENDAR'); return lines.join('\r\n'); } — node:* 외 의존 0.
- src/store.mjs icsForAssignee({assignee=null,project=null}={}) { const items=this.items({project}).filter(i=>i.due && (!assignee || i.assignee_ref===assignee)); return this.icsForItems(items,{calName: assignee?`dev-erp:${assignee}`:'dev-erp'}); }.
- server.mjs: 'if (path === "/api/calendar.ics") { const ics=store.icsForAssignee({ assignee: qp.assignee||null, project: qp.project||null }); store.appendEvent({ actor_ref:"owner", actor_kind:"human", kind:"ics_export", used_refs:["items"], data_label:"meta" }); return send(res, 200, ics, "text/calendar", { "content-disposition":"attachment; filename=\"dev-erp.ics\"" }); }' — send 가 type!=='application/json' 이면 본문 그대로(기존 send 시그니처 L59 호환).
- static/app.js: hubSchedule(U-1c) 및 deadline_cal 팝아웃에 '<a class="fav-chip" href="/api/calendar.ics?project=${encodeURIComponent(p.id)}" download>${L.ics_export}</a>'; 홈 deadline_cal 위젯 하단 전체 '내 일정 ICS' 링크(assignee 미지정=전체).
