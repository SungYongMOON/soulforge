# apps/skin-lab-storybook

## 목적

- `skin-lab-storybook` 는 renderer shell 과 분리된 skin playground 다.
- fixture-derived story state 와 theme package 만으로 material/token 변화를 병렬 개발한다.

## 포함 대상

- theme selector
- story selector
- isolated preview states
- Vite preview shell

## 제외 대상

- canonical source read
- integration bridge
- persistence
