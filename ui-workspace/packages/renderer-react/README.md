# packages/renderer-react

## 목적

- `renderer-react` 는 renderer v1 의 React presentational layer 를 둔다.
- renderer-core 의 normalized state 와 view-model 을 받아 Adventurer's Desk surface 로 렌더링한다.

## 포함 대상

- React renderer surface component
- local selection state
- renderer structural CSS
- theme-contract driven theme hook consumption

## 제외 대상

- fixture loading
- canonical integration bridge
- theme token definition
- theme package registry
