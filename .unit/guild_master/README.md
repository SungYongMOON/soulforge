# .unit/guild_master

- `guild_master/` 는 현재 사용자가 기본적으로 대화하는 guild master unit 을 둔다.
- 이 unit 은 intake, mission review, boundary review, nightly oversight, promotion handoff, delegation 성향의 administrator owner lens 를 가진다.
- `unit.yaml` 은 current-default guild master shape 를 `summary`, `identity`, `class_ids` 중심으로 고정한다.
- 이 경로의 tracked sample 은 owner surface 만 설명하고 runtime truth 는 담지 않는다.
