# .agent/export

## 목적

- `export/` 는 본체 자산의 내보내기 구조를 둔다.
- 내부 자산을 외부 형식이나 패키지로 정리하는 경계를 분리한다.

- 내보내기 규칙
- 패키징 메타와 출력 구조 정의

- 프로젝트 배포 파이프라인
- 클래스 도구 연결 구현
