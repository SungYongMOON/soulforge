# .agent/identity

## 목적

- `identity/` 는 본체의 정체성과 기본 식별 정보를 둔다.
- 클래스나 프로젝트에 종속되지 않는 식별 계층을 분리한다.

## 포함 대상

- 본체 식별 메타
- 정체성 관련 기본 선언과 참조 정보

## 제외 대상

- `class.yaml`, `loadout.yaml` 같은 클래스 메타
- 프로젝트별 식별자와 계약 정보
