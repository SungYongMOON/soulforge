# .agent/memory

## 목적

- `memory/` 는 본체의 장기 기억을 둔다.
- 클래스 교체 후에도 유지되어야 하는 기억을 이 경계 안에서 관리한다.

- 장기 기억 자산
- 기억 색인과 유지 정책에 필요한 메타

- 설치형 지식 팩
- 프로젝트 산출물 원본
