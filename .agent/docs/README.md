# .agent/docs

## 목적

- `.agent/docs/` 는 본체 소유 문서를 둔다.
- 루트 `docs/` 와 분리해서 body owner 문서를 관리한다.

## 포함 대상

- 본체 구조 설명 문서
- 본체 owner 기준의 메모와 규약 문서
- `body.yaml` 과 `body_state.yaml` 을 설명하는 메타 계약 문서

## 제외 대상

- 저장소 전체 구조 문서
- 클래스 owner 문서와 프로젝트 전용 문서
