# .agent/docs/architecture

## 목적

- `.agent/docs/architecture/` 는 `.agent` legacy bridge 문서를 잠시 보관하는 경로다.
- active owner-local 구조 문서는 이제 `.registry/docs/architecture/` 를 기준으로 본다.

## 포함 대상

- cleanup 전까지 필요한 migration note
- legacy 설명 문서의 임시 보관본

## 제외 대상

- active `.registry` owner 문서
- 새 canonical species/hero 규칙 본문

## 규칙

- Stable
- `.agent` 는 active root 가 아니므로 새 canonical 규칙을 여기서 고정하지 않는다.
- 문서가 필요하면 `.registry/docs/architecture/` 와 관계를 먼저 본다.
