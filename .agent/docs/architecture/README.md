# .agent/docs/architecture

## 목적

- `.agent/docs/architecture/` 는 본체 구조 문서의 정본 위치다.
- 본체 계층의 책임과 경계를 루트 문서와 분리해서 관리한다.

## 포함 대상

- `AGENT_BODY_MODEL.md` 같은 본체 구조 문서
- 본체 owner 기준의 구조 규약
- `BODY_METADATA_CONTRACT.md` 같은 본체 메타 규약

## 제외 대상

- 저장소 전체 아키텍처 문서
- 클래스 구조 문서와 프로젝트 연결 문서
