# hero overlay 모델

## 목적

- hero 를 `.agent/identity` 의 optional overlay 로 고정한다.
- hero 와 profile, workflow, species 의 경계를 분명히 한다.

## 규칙

- hero 는 제거 대상이 아니다.
- hero 는 class 로 내려가지 않는다.
- hero 는 profile 대체재가 아니다.
- hero 는 species 위에 얹히는 optional identity overlay 다.
- hero candidate 는 실존 인물이나 특정 IP 캐릭터의 직접 복제본이 아니라 motif 또는 imprint 로 정규화한다.

## 작용 범위

- communication tone bias
- question style bias
- memory recall bias
- autonomic reminder signature
- artifact signature
- workflow selection bias

## 작용하지 않는 범위

- factual validation rules
- policy floor
- installed asset availability
- workflow required semantics

## 관계

- workflow 는 explicit required 조합식이다.
- profile 은 default preference mode 다.
- hero 는 애매할 때 어떤 결로 기우는지를 정하는 bias 다.
- species 는 durable default 체질이다.
