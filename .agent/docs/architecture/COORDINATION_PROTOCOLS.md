# coordination protocols

## 목적

- `protocols/` 가 팀 협업과 body operating contract 에서 맡는 역할을 정의한다.

## 범위

- request, handoff, decision, incident, escalation contract 를 다룬다.

## 포함 대상

- protocol role
- contract 경계
- communication/policy/runtime/sessions 와의 연결

## 제외 대상

- 일반 human conversation rule
- shared team playbook 본문

## 미래 확장 방향

- shared protocol 표준이 생기면 `_teams/shared` 아래로 확장하고 body private default 는 여기서 유지한다.
