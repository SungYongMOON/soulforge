# .agent/engine

## 목적

- `engine/` 는 본체의 실행 기반과 엔진 설정을 둔다.
- 클래스별 장비가 아니라 본체 공통 실행 기반을 다룬다.

## 포함 대상

- 본체 엔진 설정
- 실행 기반 메타와 기본 런타임 연결 정의

## 제외 대상

- 도구별 실행 래퍼
- 프로젝트별 실행 스크립트와 워크플로우
