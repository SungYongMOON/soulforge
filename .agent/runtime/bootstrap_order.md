# runtime bootstrap order

1. policy floor
2. identity defaults
3. registry bindings and indexes
4. communication profiles
5. protocols contracts
6. sessions continuity
7. memory recall
8. artifacts lookup
9. autonomic checks

runtime은 이 순서를 기준으로 기관을 읽되, 관측되지 않은 actual runtime state 를 추정해 삽입하지 않는다.
