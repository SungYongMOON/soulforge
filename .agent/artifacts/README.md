# .agent/artifacts

## 목적

- `artifacts/` 는 본체 소유 산출물을 둔다.
- 프로젝트 현장 산출물과 구분되는 본체 측 결과물을 관리한다.

## 포함 대상

- 본체가 생성한 공용 산출물
- 본체 산출물 메타와 보관 구조

## 제외 대상

- 프로젝트별 결과물 원본
- 클래스 문서와 클래스 지식 팩
