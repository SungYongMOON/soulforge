# .agent/policy

## 목적

- `policy/` 는 본체의 정책과 안전 규칙을 둔다.
- 클래스나 프로젝트보다 상위에 있는 본체 기본 제약을 관리한다.

## 포함 대상

- 본체 정책 문서
- 안전 규칙과 허용 범위 메타

## 제외 대상

- 클래스 운영 절차
- 프로젝트 계약과 프로젝트별 예외 규칙
