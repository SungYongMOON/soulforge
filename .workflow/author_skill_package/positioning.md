# author_skill_package positioning

- `author_skill_package` 는 현재 Soulforge skill authoring lane 의 canonical sample 이자 current default workflow 다.
- `guild_master_cell` 과 `guild_master_01` 도 같은 lane 의 canonical sample 이자 current default composition / unit 이다.
- 이 셋은 지금의 기본 authoring lane 으로 쓰지만, future skill authoring path 전체의 universal operating standard 로 확정한 것은 아니다.
