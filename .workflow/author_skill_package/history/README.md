- `history/` 는 skill package authoring lane 에서 public-safe 로 남길 수 있는 curated lessons 만 둔다.
- raw authoring transcript, local install logs, project-specific smoke check dumps 는 `_workmeta/<project_code>/runs/<run_id>/` 아래에 남긴다.
- public-safe history 는 package draft 가 어떤 boundary 판단으로 생겼는지, 그리고 install handoff 직전까지 무엇이 확정됐는지 요약하는 용도로 쓴다.
- upstream import 사례에서는 source package 성격, portable bundle 범위, local-only runtime input 분리를 함께 기록한다.
