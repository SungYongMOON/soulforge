# Outbound Mail Authoring Calibrations

This folder is reserved for future public-safe synthetic calibration fixtures.

Allowed calibration material:

- synthetic mail authoring requests
- synthetic project keyword examples
- redacted footer-presence checks
- public-safe evaluator scorecards

Forbidden material:

- real mail bodies
- real recipient lists
- raw Outlook exports
- attachment payloads
- exact footer contact values
- full company security disclaimer text
- secrets or local runtime paths
