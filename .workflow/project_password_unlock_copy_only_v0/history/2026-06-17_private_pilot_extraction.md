# 2026-06-17 Private Pilot Extraction

This workflow was extracted from an owner-approved private project cleanup run.
The run established the reusable pattern: project context binding, project-local
password candidate file setup, route dry-run, copy-only smoke execution,
copy-only batch execution, original-integrity verification, metadata-only
reporting, and owner handoff for any later project tree replacement.

The public workflow package intentionally omits project names, source filenames,
password values, document text, local absolute paths, and raw output files.
