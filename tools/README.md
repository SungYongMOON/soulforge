# tools

## 목적

- `tools/` 는 renderer 와 root-owned 검증용 보조 도구를 둔다.
- 앱이나 reusable package 바깥에서 저장소 정합성을 검사하는 lint/validator 를 관리한다.

## 포함 대상

- `ui-lint/`

## 제외 대상

- canonical source 의 정본 정의
- renderer runtime package
